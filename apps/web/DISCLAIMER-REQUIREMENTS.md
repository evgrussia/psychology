# Требования к использованию дисклеймера

Этот документ описывает требования к использованию компонента `Disclaimer` на страницах приложения согласно техспецификации FEAT-WEB-03.

## Обязательное использование дисклеймера

Компонент `Disclaimer` должен использоваться на следующих типах страниц:

### 1. Интерактивные страницы (Quizzes, Tests, Interactive Modules)

**Где:** На страницах результатов интерактивов (`/interactive/quizzes/{slug}/`, `/interactive/navigator/`, и т.д.)

**Как использовать:**
```tsx
import { Disclaimer } from '../../../design-system/components';

<Disclaimer variant="info" showEmergencyLink title="Важная информация">
  Результат носит информационный характер и не является медицинским диагнозом. 
  Если вам сейчас очень тяжело или есть риск причинить вред себе/другим — 
  пожалуйста, обратитесь за экстренной помощью.
</Disclaimer>
```

### 2. Страницы booking (Запись на консультацию)

**Где:** На страницах записи (`/booking/`, `/booking/service/`, `/booking/slot/`, и т.д.)

**Как использовать:**
```tsx
import { Disclaimer } from '../../../design-system/components';

<Disclaimer variant="info" showEmergencyLink title="Важно понимать">
  Психологическая консультация не является медицинской услугой. 
  Если вы находитесь в остром кризисном состоянии, пожалуйста, 
  обратитесь в специализированные службы.
</Disclaimer>
```

### 3. Контентные страницы

**Где:** На страницах контента, где это применимо (например, `/how-it-works/`)

**Статус:** ✅ Уже реализовано в `PageClient.tsx`

## Компонент Disclaimer

Компонент находится в `design-system/components/Disclaimer/Disclaimer.tsx`

**Props:**
- `variant`: `'warning' | 'info'` - вариант стиля
- `title`: `string` - заголовок (по умолчанию "Важное уведомление")
- `showEmergencyLink`: `boolean` - показывать ли ссылку на экстренную помощь
- `children`: `React.ReactNode` - содержимое дисклеймера

**Пример использования:**
```tsx
<Disclaimer variant="info" showEmergencyLink title="Важная информация">
  Текст дисклеймера
</Disclaimer>
```

## Ссылка на экстренную помощь

Все дисклеймеры на интерактивных и booking страницах должны включать ссылку на экстренную помощь:
- Используйте `showEmergencyLink={true}`
- Ссылка ведет на `/emergency`

## Требования техспецификации

Согласно FEAT-WEB-03:
- **G2:** Единый компонент дисклеймера для интерактивов/контента/booking
- **AC-3:** Компонент дисклеймера переиспользуется в интерактивах/контенте/booking

## Статус реализации

- ✅ Компонент `Disclaimer` реализован
- ✅ Используется на главной странице (`HomeClient.tsx`)
- ✅ Используется на странице "Как проходит консультация" (`PageClient.tsx`)
- ⏳ Должен быть добавлен при создании страниц booking
- ⏳ Должен быть добавлен при создании страниц интерактивов

## Примечания

Когда создаются новые страницы booking или интерактивов, обязательно:
1. Импортировать компонент `Disclaimer`
2. Добавить дисклеймер с соответствующим текстом
3. Использовать `showEmergencyLink={true}` для ссылки на `/emergency`
