## `/start-project <идея>`

### Назначение
Старт нового проекта по агентской системе: создать Project Brief и запустить Discovery.

### Ожидаемые артефакты
- `context/project-brief.yaml`
- (опционально) `docs/README.md` + каркас `docs/*` (если пользователь просит)

### Минимальный ответ
- краткое понимание идеи
- 5–10 уточняющих вопросов (если критично)
- следующий шаг: `/route product ...` + `/route research ...` + `/route analytics ...`
