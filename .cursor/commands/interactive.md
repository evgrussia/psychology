# /interactive <module-id>

## Назначение
Показать спецификацию интерактивного модуля из Interactive-Modules-Matrix.md

## Формат
```
/interactive <module-id>
```

## Интерактивные модули проекта

### Квизы и диагностики
| ID | Название | Тема |
|----|----------|------|
| `QZ-01` | Экспресс-диагностика тревожности | Тревога |
| `QZ-02` | Тест на выгорание | Выгорание |
| `QZ-03` | Оценка качества отношений | Отношения |

### Генераторы и инструменты
| ID | Название | Функция |
|----|----------|---------|
| `GEN-01` | Генератор "Первый шаг" | Персонализированные рекомендации |
| `BND-01` | Генератор скриптов границ | Фразы для границ |
| `RES-01` | Термометр ресурса | Оценка энергобюджета |

### Дневники
| ID | Название | Формат |
|----|----------|--------|
| `DR-01` | Дневник эмоций | Ежедневный трекинг |
| `DR-02` | ABC-дневник | Когнитивный анализ |

### Челленджи
| ID | Название | Длительность |
|----|----------|--------------|
| `CH-01` | 7 дней осознанности | 7 дней |
| `CH-02` | Неделя заботы о себе | 7 дней |

### Навигатор
| ID | Название | Тип |
|----|----------|-----|
| `NAV-01` | Навигатор состояния | Ветвящийся сценарий |

## Примеры использования

```
/interactive QZ-01
```
→ Показывает: вопросы, логика подсчёта, уровни результатов, кризисная обработка

```
/interactive DR-01
```
→ Показывает: структура записи, поля, валидация, privacy требования
