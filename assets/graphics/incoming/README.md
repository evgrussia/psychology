# Incoming (временная папка)

Сюда можно **временно положить** сгенерированную графику (как она у вас сейчас экспортирована),
а затем мы разложим её по целевым папкам:

- `assets/graphics/hero/`
- `assets/graphics/spot/`
- `assets/graphics/icons/`
- `assets/graphics/patterns/`
- `assets/graphics/photos/`
- `assets/graphics/abstract/`

Если у файлов нет нейминга по конвенции — это нормально: я переименую и обновлю `manifest.json`.
