## 1. Источник требований к экранам

В **FIGMA_CONCEPT_PROMPT.md** перечислены **не названия экранов**, а:

- философия дизайна (спокойствие, забота, доверие, бережность, надежда);
- визуальный стиль (цвета, типографика, spacing, mobile-first);
- компоненты и паттерны (карточки, кнопки, формы, навигация);
- кризисные элементы (экстренная помощь всегда доступна и заметна).

Документ отсылает к **FIGMA_MAKE_PROMPTS.md** для верстки конкретных экранов; в репозитории присутствует только **FIGMA_CONCEPT_PROMPT.md**. Список экранов для аудита собран из:

- существующих компонентов в `frontend/src/app/components/`;
- маршрутов в `frontend/src/app/App.tsx`;
- области работ из `context/project-brief.yaml` (главная, контент, запись, кабинет, интерактивы, юридическое, кризис).

## 3. Соответствие вёрстки концепции FIGMA_CONCEPT_PROMPT.md

### 3.1 Проверенные критерии

- **Цвета:** Primary #A8B5FF, Secondary #FFD4B5, Accent #C8F5E8, текст #2D3748 / #718096, семантика #7FD99A, #FFC97F, #FF9A9A (мягкие).
- **Стиль:** градиенты, скругления 16–24px (rounded-2xl/3xl), при необходимости backdrop-blur.
- **Mobile-first:** отступы `px-4 sm:px-6 lg:px-8`, адаптивные размеры текста и кнопок.
- **Кризис:** на EmergencyHelpPage и QuizCrisisPage — заметные контакты, мягкие цвета (не агрессивный красный), без продающих CTA.

### 3.2 Результат проверки по файлам

По всем **25** файлам страниц (включая Quiz* и NavigatorStartPage) в коде используются:

- палитра из концепции (#A8B5FF, #FFD4B5, #C8F5E8, #2D3748, #718096 и семантика);
- градиенты и скругления;
- единообразные отступы и адаптивные классы.

Выборочно проверены:

- **HomePage** — hero с градиентом, glassmorphism-карточки, CTA 44px+, mobile-first.
- **EmergencyHelpPage** — экстренная помощь выделена, телефон доверия, мягкие оттенки (#FFB5C5, #FFD4B5).
- **QuizStartPage** — цвета и типографика по концепции, описание опросника.
- **QuizCrisisPage** — кризисный блок с контактами, мягкий коралловый (#FF9A9A), без агрессивного красного.
- **NavigatorStartPage** — палитра эмоций из концепции, карточки и кнопки в стиле платформы.
- **BookingServicePage**, **TopicsHubPage** — градиенты карточек, скругления, консистентность с концепцией.

**Вывод:** Все сверстанные экраны в `frontend/src/app/components` соответствуют концепции из FIGMA_CONCEPT_PROMPT.md (цвета, типографика, mobile-first, кризисные элементы).
