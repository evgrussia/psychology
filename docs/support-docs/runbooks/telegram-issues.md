# Runbook: Проблемы с Telegram (Telegram Issues)

Интеграция с Telegram обеспечивает уведомления и планы практик.

## 1. Пользователь не получает уведомления
**Действия:**
1. Проверить, подключен ли Telegram в профиле пользователя (наличие `telegram_id`).
2. Убедиться, что бот не заблокирован пользователем.
3. Проверить логи Celery-задач на отправку сообщений.
4. Проверить статус Telegram Bot API (бывают глобальные сбои).

## 2. Бот не реагирует на команды
**Действия:**
1. Проверить, запущен ли процесс бота (или webhook-обработчик).
2. Проверить логи на наличие ошибок `Unauthorized` (истек токен) или `Conflict` (запущено два экземпляра бота).
3. Убедиться, что вебхуки от Telegram доходят до нашего сервера.

## 3. Ошибка связки аккаунтов (Deep Linking)
**Симптомы:** При переходе с сайта в бот не происходит автоматическая авторизация.
**Действия:**
1. Проверить генерацию ссылки с токеном в Backend.
2. Убедиться, что токен не просрочен (время жизни обычно 5-10 минут).
3. Проверить логи обработки команды `/start [token]` в боте.
