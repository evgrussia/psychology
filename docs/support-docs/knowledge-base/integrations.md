# Внешние интеграции (Integrations)

Система «Эмоциональный баланс» взаимодействует с несколькими внешними сервисами.

## 1. Google Calendar API
- **Назначение:** Синхронизация расписания специалиста и создание событий для встреч.
- **Критичность:** Высокая. Без неё невозможна корректная запись.
- **Типовые проблемы:** Истечение срока OAuth-токена, достижение лимитов API.

## 2. ЮKassa API
- **Назначение:** Приём платежей и выплаты.
- **Критичность:** Высокая.
- **Схема работы:** Система создаёт платёж → пользователь оплачивает на стороне ЮKassa → ЮKassa шлёт вебхук нам → мы подтверждаем запись.

## 3. Telegram Bot API
- **Назначение:** Рассылка уведомлений и работа с планами практик.
- **Критичность:** Средняя (основной функционал сайта работает без него).
- **Механизм:** Работает через Webhooks для мгновенной реакции на действия пользователя.
