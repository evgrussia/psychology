# FEAT-WEB-11 Implementation Report

Дата: 2026-01-17  
Статус: ✅ выполнено

## Что реализовано
- Обновлена структура главной страницы: добавлен блок "быстрого старта", переставлены секции для раннего доступа к первому шагу и доверительным блокам.
- Улучшена микрокопия CTA и заголовков для снижения трения и повышения ясности выбора (консультация vs Telegram).
- Добавлен обратимый флаг `homepage_conversion_v2_enabled` для быстрого отката.
- Доработан трекинг CTA: `cta_click` включает `cta_label`, `cta_tg_click` включает `cta_id` для аналитики источников.

## Соответствие AC/DoD
- **UX/A11y**: структура и тексты улучшают ясность выбора, семантика не нарушена.
- **Измеримость**: CTA события продолжают фиксироваться согласно `docs/Tracking-Plan.md`.
- **Обратимость**: изменения изолированы флагом `homepage_conversion_v2_enabled`.

## Ключевые изменения в коде
- Главная страница: `apps/web/src/app/HomeClient.tsx`.
- Feature flags: `apps/web/src/lib/feature-flags.ts`, тесты `apps/web/src/lib/feature-flags.spec.ts`.

## Как проверить
1. Открыть главную страницу `/` и убедиться, что:
   - есть блок "быстрого старта";
   - секции идут в новом порядке;
   - CTA тексты обновлены.
2. Выключить флаг `NEXT_PUBLIC_FF_HOMEPAGE_CONVERSION_V2=false` и проверить, что вернулась предыдущая структура.
3. Нажать CTA и убедиться, что события отправляются (`cta_click`, `cta_tg_click`).
