## 1) Summary
AI‑агенты (после релиза 1): навигация “следующий шаг”, консьерж записи, контент‑помощник только human‑in‑the‑loop. Жёсткие safety/guardrails: не диагноз, не терапия в чате, кризис‑режим, минимизация данных.

## 2) Goals (высокоуровнево)
- агент не получает P2 тексты без отдельного согласия,
- crisis detector “перекрывает” ответы,
- все ответы — “информационная навигация”, не клинические советы.

---

## Appendix: Implementation stages (чтобы уложиться в контекст)

1. **Stage A (~120k):** Safety layer + crisis classifier + запреты/редакция данных + eval harness.
2. **Stage B (~120k):** Агент-навигация “следующий шаг” на сайте (без хранения текста).
3. **Stage C (~120k):** Агент-консьерж записи (только сервисные ответы, без терапии) + human handoff.
