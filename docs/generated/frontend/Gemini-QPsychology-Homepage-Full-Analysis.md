# Полный визуальный и функциональный анализ главной страницы Q Psychology

Этот документ содержит детальное описание визуальных стилей, эффектов, поведения элементов и структуры главной страницы [qpsychology.com.au](https://www.qpsychology.com.au/), дополняющее и расширяющее предыдущие наблюдения.

## 1. Цветовая палитра и Типографика

*   **Основной цвет (Brand Primary)**: Глубокий тёмно-зелёный (Forest Green). Используется для фонов прелоадера, активных элементов, текста на светлом фоне.
*   **Фоновый цвет (Background)**: Светло-кремовый / Off-white. Создает мягкий, "органический" контраст с тёмно-зелёным, избегая стерильности чистого белого.
*   **Типографика**: Современный гротеск (Sans-serif). Заголовки крупные, чистые. Текст читабельный, с достаточным "воздухом" (letter-spacing, line-height).

## 2. Загрузка (Preloader)

*   **Экран**: Полноэкранная заливка основным тёмно-зелёным цветом.
*   **Элементы**: По центру логотип "Q" и название "PSYCHOLOGY".
*   **Поведение**: Логотип отображается до полной загрузки скриптов/стилей, затем экран плавно исчезает (fade-out), открывая контент.

## 3. Хедер (Header)

Хедер является одним из ключевых элементов дизайна, выполнен в стиле "плавающей капсулы".

### Общие свойства
*   **Форма**: "Capsule" / "Pill" (сильное скругление углов, border-radius ~50px или больше).
*   **Позиционирование**: `Sticky` (фиксируется у верхней границы экрана при скролле).
*   **Эффекты**: Мягкая тень (`box-shadow`), создающая ощущение левитации над контентом.
*   **Z-Index**: Всегда поверх остального контента.

### Десктопная версия (Desktop)
В отличие от мобильной версии, хедер на десктопе содержит развернутую навигацию:
*   **Логотип**: Слева ("Q Psychology").
*   **Центральное меню**: Горизонтальный ряд ссылок:
    *   `About`
    *   `Team`
    *   `Services` (с выпадающим списком/dropdown)
    *   `Fee`
    *   `Blog`
    *   `Locations` (с выпадающим списком/dropdown)
    *   `Contact`
*   **Правая часть**:
    *   Поле поиска (раскрывающееся или иконка).
    *   Кнопка "Menu" (Гамбургер) — для открытия полного оверлея, дублирующего навигацию.

### Мобильная версия (Mobile)
*   **Состав**: Логотип (слева) + Иконка Гамбургера (справа).
*   **Скрытие**: Горизонтальные ссылки скрыты в бургер-меню.

## 4. Оверлей-меню (Overlay Navigation)

Открывается по клику на кнопку "Menu" (Гамбургер).

*   **Тип**: Полноэкранное или крупное модальное окно.
*   **Фон**: Светлый (кремовый).
*   **Анимация кнопки**: Иконка "Гамбургер" плавно превращается в "X" (Close).
*   **Структура пунктов**:
    *   Крупная типографика.
    *   Разделители (тонкие горизонтальные линии) между пунктами.
*   **Интерактивность**:
    *   Пункты с вложенностью (`Services`, `Locations`) имеют иконку **Chevron Down**.
    *   **Mobile**: Работают как **Аккордеон** (раздвигают список вниз по тапу).
    *   **Desktop**: Могут работать как ховеры или клик-аккордеоны.

## 5. Hero-секция (Первый экран)

*   **Фон**: Полноэкранное фото (Full-screen image). Используется ротация изображений (слайдер) или случайная загрузка атмосферных фото природы/людей.
*   **Оверлей**: Затемнение (Dark overlay) поверх фото для контрастности белого текста.
*   **Контент**: Крупный заголовок, вводный текст.
*   **Навигация**: Элемент **"Scroll to explore"** внизу экрана — текстовая подсказка с анимацией, приглашающая к прокрутке.

## 6. UI-Компоненты и Интерактивность

### Кнопки (CTA Buttons)
Фирменный стиль кнопок (например, *"Reach out to us"*, *"More about us"*):
*   **Форма**: Комбинированная "Пилюля".
    *   Текстовая часть в овале.
    *   Справа примыкает **круг** (circle) с иконкой **стрелки**.
*   **Ховер (Desktop)**:
    *   Микро-анимация стрелки (сдвиг вправо-влево или по диагонали).
    *   Изменение цвета фона/заливки (инверсия или затемнение).

### Списки услуг (Service Links)
Секция *"Our services..."*:
*   **Вид**: Крупные горизонтальные блоки-строки.
*   **Структура**: Название услуги (слева) + Иконка круглой стрелки (справа).
*   **Разделители**: Тонкие линии между строками.
*   **Ховер**: При наведении строка может подсвечиваться, стрелка анимироваться. Ссылки ведут на детальные страницы услуг.

### Аккордеоны (Accordions)
Используются для компактного отображения информации (например, *"We’re Moving"* или список локаций):
*   **Хедер аккордеона**: Заголовок + Иконка `+` (Плюс).
*   **Состояние**:
    *   Закрыт: Виден только заголовок.
    *   Открыт: Иконка `+` меняется на `-` (или поворачивается на 45° в `x`), контент плавно выезжает вниз.
*   **Внутренний контент**: Текст, списки (ul/li), ссылки.

## 7. Футер (Footer)

Секция в самом низу страницы (тёмный или светлый фон, контрастный к основной части).

*   **Социальные сети**: Иконки-ссылки (Instagram, Facebook, LinkedIn и т.д.).
*   **Юридическая информация**:
    *   `Privacy Policy` (Политика конфиденциальности).
    *   `Staff resources` (Вход для сотрудников).
*   **Копирайт**: `Site by Heads Under™` (Авторство разработки).
*   **Логотип**: Часто дублируется в монохромном варианте.

## 8. Адаптивность и Поведение (Responsive Behavior)

| Элемент | Desktop | Mobile/Tablet |
| :--- | :--- | :--- |
| **Хедер** | Полное меню + Поиск + Бургер | Только Лого + Бургер |
| **Взаимодействие** | Hover-эффекты активны | Tap-эффекты (мгновенный отклик) |
| **Вложенные меню** | Выпадающие списки (Dropdowns) | Аккордеоны (раздвигание контента) |
| **Сетка (Grid)** | Многоколоночная (текст + фото) | Одноколоночная (блоки друг под другом) |

## 9. Динамика переходов

*   **Smooth Scroll**: Плавная прокрутка к якорям.
*   **Fade-in**: Элементы контента могут плавно появляться при скролле (lazy load effects).
*   **Page Transitions**: Плавные переходы между страницами (отсутствие резкого белого экрана).
