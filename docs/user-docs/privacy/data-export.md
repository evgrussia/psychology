# Экспорт данных

Ваши данные принадлежат вам. Мы предусмотрели возможность выгрузки информации, чтобы вы могли использовать её вне сервиса или сохранить для личного архива.

## Что можно экспортировать

### 1. Дневниковые записи
Все ваши заметки, мысли и наблюдения из разделов самопомощи можно выгрузить в структурированном виде.

### 2. Результаты тестов и диагностик
Ваши ответы и полученные ориентиры из интерактивных модулей.

### 3. История встреч
Список ваших консультаций с указанием дат и тем (если вы их указывали).

## Как выполнить экспорт
1. Перейдите в Личный кабинет.
2. Найдите раздел «Экспорт данных» или «Мои данные».
3. Выберите формат:
   - **PDF**: красиво оформленный документ, удобный для чтения или пересылки специалисту.
   - **JSON**: технический формат, если вы хотите перенести данные в другую систему (для продвинутых пользователей).
4. Нажмите «Сформировать файл». Через несколько секунд ссылка на скачивание будет готова.

## Подготовка к консультации
Особая функция — «Подготовка к встрече». Она собирает последние важные записи и результаты квизов в один файл. Это помогает вам структурировать запрос, а специалисту — быстрее понять вашу ситуацию.
