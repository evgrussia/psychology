# Навигация по сервису

Мы постарались сделать интерфейс «Эмоционального баланса» простым и поддерживающим, чтобы вы могли сосредоточиться на своём состоянии, а не на поиске кнопок.

## Основные разделы

### 1. Главная страница
Здесь собраны все точки входа: быстрые упражнения, новые статьи и возможность записаться на встречу. Это ваш «центр управления» спокойствием.

### 2. С чего начать
Если вы не знаете, что именно вам сейчас нужно — загляните в этот раздел. Мы предложим вам:
- **Навигатор состояния**: поможет выбрать подходящую практику за 1–2 минуты.
- **Мини-диагностики**: короткие тесты-ориентиры.
- **Первый шаг**: простое упражнение, которое можно сделать прямо сейчас.

### 3. Библиотека ресурсов
Здесь хранятся статьи, аудио-практики и чек-листы. Вы можете искать их по темам (например, «Границы», «Тревога», «Ресурс») или по времени, которое у вас есть.

### 4. Личный кабинет
Ваше приватное пространство. Здесь находятся:
- История ваших записей на встречи.
- Дневники самочувствия.
- Материалы, которыми поделился с вами специалист.

## Поиск
В верхней части сайта есть поиск по ключевым словам. Он поможет быстро найти нужную статью или упражнение.

## Telegram-бот
Сайт тесно связан с нашим Telegram-ботом. Он помогает получать напоминания о встречах и короткие планы практик на неделю, чтобы поддержка всегда была под рукой.
