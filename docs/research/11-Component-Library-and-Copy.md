# 11 — Библиотека компонентов и микрокопирайтинг (для всех модулей)

Документ нужен, чтобы:
- быстро собирать страницы из повторяемых блоков,
- не “ломать” тон (эмпатия, безопасность, без давления),
- иметь готовые тексты для состояний UI (ошибки/пусто/успех).

## 1) Глобальные UI‑компоненты (layout)
### 1.1 Header (шапка)
- **Элементы**: лого, меню, кнопка «Запись», (опционально) «С чего начать»
- **Состояния**: обычное / липкое / мобильное меню

### 1.2 Footer (подвал)
- **Блоки**:
  - навигация
  - «экстренная помощь» (ссылка)
  - политика/согласия/оферта/дисклеймер
  - контакты

### 1.3 Trust Strip (полоса доверия)
Короткие факты:
- «конфиденциально»
- «бережно и без давления»
- «прозрачные правила и границы»
- «можно начать с маленького шага»

## 2) Компоненты «путь клиента» (главная/лендинги)
### 2.1 Problem Card (карточка состояния)
- **Props**: title, shortDescription, timeToBenefit, link
- **Варианты**: тревога / выгорание / отношения / самооценка / границы

### 2.2 Start Step Selector (выбор “с чего начать”)
- **Состояния**: выбор состояния → выбор формата (тест/упражнение/статья)

### 2.3 CTA Block (универсальный)
- **Типы CTA**:
  - primary: «Записаться»
  - secondary: «Получить план в Telegram»
  - tertiary: «Попробовать упражнение»
- **Правило**: у каждого CTA есть 1 строка “зачем”

Пример:
- Кнопка: «Получить план в Telegram»
- Подпись: «Пришлю 3 коротких шага и напоминание — без спама»

### 2.4 How It Works (как проходит консультация)
- **Компоненты**: шаг 1–2–3, длительность, формат, что будет на первой встрече

## 3) Компоненты интерактивов (квизы/тесты/генератор)
### 3.1 Quiz Intro
- title, duration, disclaimerShort, startButton

### 3.2 Quiz Question
- тип: шкала/выбор
- подсказка «можно пропустить» (если допускаем)

### 3.3 Quiz Progress
- шаги, процент, “сколько осталось”

### 3.4 Quiz Result Card
- **Секции**:
  - уровень + короткое объяснение
  - «шаги сейчас» (2–3)
  - «на неделю» (2–3)
  - «когда стоит обратиться к специалисту»
  - CTA (Telegram/запись)

### 3.5 Crisis Banner (экстренная помощь)
См. `01-IA-and-UserJourneys.md`.

### 3.6 First Step Generator
- выбор состояния (chips)
- карточка упражнения (таймер/аудио)
- кнопка «не подходит → другой вариант»

### 3.7 State Navigator (ветвящийся сценарий)
- **Компоненты**:
  - Intro (время, дисклеймеры, кнопка “начать”)
  - Step (варианты выбора 2–3, подсказка “нет правильного ответа”)
  - Navigator Progress (шаг/всего или прогресс‑бар)
  - Result (профиль результата + шаги “сейчас” + “на неделю” + CTA)
- **Состояния**: loading (если подтягиваем контент), back, skip (если допускаем), crisis mode

### 3.8 Resource Thermometer (энергобюджет дня)
- **Компоненты**:
  - Slider (energy/tension/support)
  - Level Badge (`resource_level`: low/moderate/high)
  - Recommendations List (2–4 пункта)
  - CTA: rituals / telegram / booking
- **Состояния**: low/moderate/high, reset, save

### 3.9 Boundaries Script Generator
- **Компоненты**:
  - Scenario Selector (work/family/partner/friends)
  - Tone Selector (soft/short/firm)
  - Goal Selector (say_no/ask/define_rule/pause)
  - Script Card (вариант фразы)
  - Copy Button + Toast
  - “More variants” (перелистывание)
- **Запрет**: хранить/треκать текст фраз — используем `variant_id`

### 3.10 Consultation Prep Builder
- **Компоненты**:
  - Wizard Stepper (5–7 шагов)
  - Multi‑select Cards (варианты)
  - Optional Textarea (необязательно)
  - Draft Summary (структура из блоков)
  - Export Buttons (copy/pdf/telegram)
- **Приватность**: если есть textarea — по умолчанию не сохранять (или хранить локально) и явно объяснять это.

### 3.11 Mini Ritual Timer
- **Компоненты**:
  - Ritual Card (название, длительность, “зачем”)
  - Timer Controls (start/pause/finish)
  - “Save to favorites”

### 3.12 Favorites / “My Kit” (избранное)
- **Компоненты**:
  - Favorites List (с группировкой по типу)
  - Empty state + CTA “С чего начать”
  - Remove action (иконка/кнопка)
  - (опционально) Sync hint “сохранить в Telegram”

## 4) Компоненты дневников и ЛК
### 4.1 Diary Entry Form
- шкалы 0–10
- триггеры (dropdown)
- поле “заметка” (опционально)
- “приватно” (переключатель видимости, если будет совместный доступ с психологом)

### 4.2 Weekly Summary
- график/сводка + 1–2 мягкие рекомендации

### 4.3 Export PDF Button
- описание: «сформировать PDF для консультации»

## 5) Компоненты записи/оплаты
### 5.1 Service Picker
- карточки услуг, что включено, длительность

### 5.2 Calendar Slot Picker
- таймзона, фильтр по формату, быстрый выбор
- состояния: нет слотов / ошибка / загрузка

### 5.3 Intake Form (короткая анкета)
- 5–8 вопросов максимум (см. `05-*`)

### 5.4 Payment Summary
- цена/депозит, политика отмены/переноса, чекбокс согласия

## 6) Компоненты доверия (профиль/отзывы)
### 6.1 Credentials Block
- образование/сертификаты + дата + (опционально) подтверждение

### 6.2 Ethics & Boundaries
- «что я гарантирую / чего не делаю»

### 6.3 FAQ Accordion
- вопросы про процесс, конфиденциальность, подготовку

### 6.4 Testimonial Card
- анонимность (инициалы/город), формат (текст/аудио)

### 6.5 Case Story (история)
- «что было → что делали → что изменилось» без деталей

## 7) Микрокопирайтинг: состояния интерфейса
### 7.1 Ошибки (бережно, без вины пользователя)
- **Сеть/сервер**: «Похоже, сейчас есть техническая пауза. Попробуйте ещё раз через минуту.»
- **Валидация**: «Кажется, здесь не хватает одного поля — давайте заполним и продолжим.»
- **Оплата**: «Платёж не подтвердился. Это бывает. Проверьте банк/попробуйте ещё раз или выберите другой способ.»

### 7.2 Пустые состояния
- **Нет слотов**: «На ближайшие дни нет свободных окон. Оставьте контакт — я напишу, когда появится время.»
- **Нет записей дневника**: «Можно начать с одного короткого пункта. 30 секунд — уже достаточно.»

### 7.3 Поддерживающие подсказки
- «Если вопрос кажется сложным — можно отвечать так, как получается сейчас.»
- «Вы можете остановиться в любой момент.»
- «Можно выбрать варианты — не нужно писать текст.»
- «По умолчанию мы не сохраняем личные тексты без вашего решения.»

## 8) Тексты CTA (готовые формулировки)
### 8.1 Запись
- «Записаться на первую встречу»
- «Разобрать ситуацию за 1 встречу»
- «Выбрать время»

Подписи:
- «Встреча 50 минут, онлайн. После записи вы получите подтверждение и инструкцию.»

### 8.2 Telegram
- «Получить план на 7 дней в Telegram»
- «Сохранить упражнение и получить напоминание»
- «Продолжение и примеры — в Telegram»

Подписи:
- «Без спама. Можно отписаться в любой момент.»

### 8.3 Уникальные интерактивы (CTA)
- «Пройти навигатор состояния»
- «Проверить ресурс на сегодня»
- «Подобрать фразу для разговора»
- «Подготовиться к первой встрече»
- «Мини‑ритуал на 3 минуты»
- «Открыть мою аптечку»

Подписи:
- «2–3 минуты. Это не диагноз — просто ориентир.»
- «Один маленький шаг — уже достаточно.»
- «Можно скопировать и использовать как есть.»

## 9) Дисклеймеры (короткие)
- «Не диагноз. Ориентир и первый шаг.»
- «Не экстренная помощь.»
- «Если стало хуже — остановитесь и сделайте паузу.»
