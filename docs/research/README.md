# Исследования проекта «Эмоциональный баланс» (индекс)

Этот набор документов — «исследовательская база», достаточная для:
- проектирования структуры портала (страницы/модули/компоненты),
- написания контента для всех ключевых разделов,
- формулирования требований к интерактивам (квизы/дневники/челленджи),
- описания безопасных воронок (запись/Telegram) без давления,
- фиксации рисков/ограничений (конфиденциальность, «не диагноз», кризисные сигналы).

## Содержание
- `01-IA-and-UserJourneys.md` — карта сайта, навигация, таксономия, пользовательские пути, контент‑модель.
- `02-Audience-Personas-JTBD.md` — сегменты, персоны, JTBD, возражения/триггеры доверия.
- `03-Content-SEO-and-Editorial.md` — контент‑стратегия, SEO‑кластеры, шаблоны страниц/статей, тон и дисклеймеры.
- `04-Interactive-Modules.md` — исследования и спецификации интерактивов: мини‑диагностики, дневники, «первый шаг», челленджи, анонимный вопрос.
- `05-Booking-Payment-ClientCabinet.md` — запись/оплата/анкеты/согласия/уведомления + личный кабинет клиента.
- `06-Telegram-Ecosystem.md` — экосистема TG: бот, выдача лид‑магнитов, сегментация, серии догрева, deep links, триггеры.
- `07-Trust-SocialProof-Profile.md` — профиль психолога, доверие, этика/границы, FAQ, отзывы/кейсы, медиа.
- `08-Admin-CRM-Analytics.md` — админ‑панель, CRM‑лиды, аналитика, шаблоны, модерация.
- `09-AI-Agents-Safety.md` — как безопасно использовать LLM/LangChain: сценарии, ограничения, кризис‑хендлинг, human‑in‑the‑loop.
- `10-Legal-Privacy-Compliance-RU.md` — комплаенс/приватность: 152‑ФЗ, политика, согласия, хранение, риски.
- `11-Component-Library-and-Copy.md` — список UI‑компонентов и библиотека микрокопирайтинга (варианты/состояния).
- `12-Community-and-Events.md` — комьюнити и мероприятия: вебинары, Q&A, клуб/подписка, групповая терапия (если включаем).
- `13-Competitive-Research-Psychology-Portals.md` — референсы похожих проектов: типы порталов, активности, контент, скрининг/диагностика, механики удержания.

## Как использовать
1) Сначала прочитайте `01-IA-and-UserJourneys.md` (это «каркас»).
2) Для наполнения страниц — `03-Content-SEO-and-Editorial.md` + `07-Trust-SocialProof-Profile.md`.
3) Для интерактивов — `04-Interactive-Modules.md` + `11-Component-Library-and-Copy.md`.
4) Для записи/Telegram — `05-*` и `06-*`.
5) Для безопасного AI и правовых вопросов — `09-*` и `10-*`.
