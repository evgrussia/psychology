# Go Live Checklist — «Эмоциональный баланс»

Этот документ содержит финальный чеклист готовности к запуску в Production.

## 1. Техническая готовность (Infrastructure)
- [ ] Production окружение настроено (Nginx, Gunicorn, systemd)
- [ ] SSL сертификаты установлены и настроены на автообновление (Let's Encrypt)
- [ ] DNS записи настроены и проверены
- [ ] CDN (Cloudflare/налоги) настроен для отдачи статики
- [ ] Все переменные окружения и секреты добавлены в Production Environment
- [ ] База данных PostgreSQL развернута и доступна
- [ ] Redis (кэш и очереди) развернут и доступен
- [ ] Firewall правила настроены (открыты порты 80, 443, 22 для whitelist)

## 2. Мониторинг и Бэкапы
- [ ] Prometheus собирает метрики со всех узлов
- [ ] Alertmanager настроен и интегрирован (Slack/PagerDuty)
- [ ] Grafana дашборды ("Golden Paths", "Errors", "Infrastructure") готовы
- [ ] Сбор логов (Filebeat/ELK/Loki) настроен и проверен
- [ ] Скрипты ежедневного бэкапа БД настроены и протестированы
- [ ] Процедура восстановления из бэкапа проверена

## 3. Функциональная готовность (Application)
- [ ] Применены все миграции базы данных
- [ ] Выполнена команда `collectstatic`
- [ ] Все P0 фичи протестированы в staging и готовы к prod
- [ ] Smoke Tests (автоматизированные) проходят успешно
- [ ] Интеграции с внешними сервисами проверены (Google Calendar, ЮKassa, Telegram)
- [ ] Email-рассылки работают (SMTP настроен)

## 4. Документация и Команда
- [ ] Документация для пользователей готова и доступна в `/docs/user-docs/`
- [ ] Документация для поддержки (Runbooks) готова в `/docs/support-docs/`
- [ ] План отката (Rollback Plan) подготовлен и проверен
- [ ] Команда деплоя на связи, роли распределены
- [ ] Окно обслуживания (Maintenance Window) согласовано и анонсировано

## 5. Безопасность
- [ ] Проведен аудит заголовков безопасности (HSTS, CSP, X-Frame-Options)
- [ ] MFA включен для всех административных аккаунтов
- [ ] Секреты не хранятся в коде или логах
- [ ] Кризисные протоколы (Crisis Safety) проверены и активны
